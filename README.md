# Orbit Clicker
Link to game: https://thucchi-cs.github.io/OrbitClicker/ 

A game that uses a single key to play! When the needle spins, you must press the space bar whenever it hits a star. If you press the space bar when the needle does not touch the star, you lose!

![Game image](<readme_img.png>)


This game was made originally by Thuc Chi Do ([thucchi-cs](https://github.com/thucchi-cs)). It uses simple HTML, CSS, and JavaScript.

Orbit Clicker was made for fun and is for Hack Club's OneKey ysws.
